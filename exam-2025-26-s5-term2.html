2. 3分
因式分解：
(a) $4x^{2} - 25$
(b) $4x^{2} - 25 - 4x - 10$
📝 答案 (點擊顯示)
(a) $(2x+5)(2x-5)$
(b) $(2x+5)(2x-5) - 2(2x+5) = (2x+5)(2x-7)$
3. 3分
令 $k$ 成為公式 $\dfrac{k}{h} + \dfrac{1}{3h} = k$ 的主項。
📝 答案 (點擊顯示)
$\dfrac{k}{h} + \dfrac{1}{3h} = k$
$\Rightarrow \dfrac{k}{h} - k = -\dfrac{1}{3h}$
$\Rightarrow k\left(\dfrac{1}{h} - 1\right) = -\dfrac{1}{3h}$
$\Rightarrow k = \dfrac{-1}{3h\left(\dfrac{1}{h} - 1\right)} = \dfrac{1}{3(1-h)}$
4. 4分
在某課室，男生人數為女生人數的 2 倍。若 10 名男生和 2 名女生離開該課室，則男生人數與女生人數相等。求在該課室原本的女生人數。
📝 答案 (點擊顯示)
設原本女生 $x$ 人，則男生 $2x$ 人。
$2x - 10 = x - 2$
$x = 8$
原本有 8 名女生。
5. 4分
(a) 解不等式 $\dfrac{x-6}{3} < x+8$
(b) 找出能同時滿足 $\dfrac{x-6}{3} < x+8$ 及 $9 - 4x \ge 25$ 的最大整數。
📝 答案 (點擊顯示)
(a) $\dfrac{x-6}{3} < x+8$
$\Rightarrow x-6 < 3x+24$
$\Rightarrow -30 < 2x$
$\Rightarrow x > -15$

(b) $9-4x \ge 25$
$\Rightarrow -4x \ge 16$
$\Rightarrow x \le -4$

同時滿足：$-15 < x \le -4$
最大整數：$-4$
6. 4分
某音樂光碟的標價為 $\$180$，並以標價的「四折」售出。
(a) 求售價
(b) 若售出該光碟的虧蝕百分率為 20%，求成本
📝 答案 (點擊顯示)
(a) 售價 = $180 \times 40\% = \$72$

(b) 設成本為 $C$
蝕 20% = 80% 的成本
$72 = C \times 80\%$
$C = 72 \div 0.8 = \$90$
7. 3分
如圖，扇形 $POQ$ 的半徑為 $15$ cm，中心角為 $72^\circ$。求扇形 $OPQ$ 的周界，答案以 $\pi$ 表示。
📝 答案 (點擊顯示)
弧長 $= 2\pi r \times \dfrac{72}{360^\circ} = 2\pi(15) \times \dfrac{1}{5} = 6\pi$
周界 $= 6\pi + 15 + 15 = 6\pi + 30$ cm
8. 4分
如圖，已知 $PQ = RS$，$\angle QPR = \angle PRS = 70^\circ$，$\angle PSR = 62^\circ$
(a) 證明 $\Delta PQR \cong \Delta RSP$
(b) 求 $\angle PQR$ 及 $\angle PRQ$
📝 答案 (點擊顯示)
(a) SAS：三邊
$PQ = RS$ (已知)
$\angle QPR = \angle PRS$ (已知)
$PR = PR$ (公共邊)
$\therefore \Delta PQR \cong \Delta RSP$

(b) $\angle PQR = \angle RSP = 62^\circ$
$\angle PRQ = 180^\circ - 70^\circ - 62^\circ = 48^\circ$
