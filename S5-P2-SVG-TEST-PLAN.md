# S5 卷二 SVG 圖像完整測試計劃

## 測試目標
驗證 Google Docs 內既 SVG 與網頁顯示既圖像是否完全一致

---

## 測試方法

### 方法一：元素數量比對

| 元素類型 | 說明 |
|----------|------|
| `<polygon>` | 多邊形/梯形 |
| `<rect>` | 矩形 |
| `<circle>` | 圓形 |
| `<line>` | 直線 |
| `<path>` | 路徑/曲線 |
| `<text>` | 文字標籤 |

### 方法二：座標比對

| 元素 | 關鍵屬性 |
|------|----------|
| 點座標 | `points="x1,y1 x2,y2 ..."` |
| 文字位置 | `x="..." y="..."` |
| 圓心 | `cx="..." cy="..."` |
| 半徑 | `r="..."` |

### 方法三：顏色比對

| 屬性 | 說明 |
|------|------|
| `stroke` | 邊線顏色 |
| `fill` | 填充顏色 |
| `stroke-width` | 線條粗細 |
