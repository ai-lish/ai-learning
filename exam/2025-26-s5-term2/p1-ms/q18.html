題目

某次全級數學測驗的標準差是 $4$。小平及小英的分數分別是 $58$ 分及 $50$ 分。已知小平的標準分是 $2$。

  1. 求全級測驗的方差及平均分。 (3 分)
  2. 求小英的標準分。 (2 分)
  3. 若老師將小英的分數剔去，小平宣稱自己的標準分會減少，你同意嗎？試解釋你的答案。 (2 分)
點擊下方卡片展開詳細解題步驟，右側圖表將同步更新。

首先處理方差 (Variance)。我們知道方差是標準差 (Standard Deviation, $\sigma$) 的平方。

已知標準差 $\sigma = 4$
方差 $= \sigma^2$
方差 $= 4^2$
方差 $\mathbf{= 16}$ 1A
常犯錯誤： 根據評分準則，方差只接受「沒有單位」或「平方分」。若誤寫為「$16$ 分」，將會被扣分！

接著求平均分 (Mean, $\bar{x}$)。利用小平的數據和標準分公式：

💡 標準分公式： $z = \frac{x - \bar{x}}{\sigma}$
$\frac{58 - \bar{x}}{4} = 2$ 1M // 列出方程

/* 兩邊同乘 4 去分母 */
$58 - \bar{x} = 8$

/* 移項化簡 */
$58 - 8 = \bar{x}$
$\bar{x} = 50$

因此，平均分 $= \mathbf{50\text{ 分}}$ 1A
考評局要求： 平均分必須清楚寫明單位「分」，若只寫數值 $50$ 沒有單位，會失去這 $1$ 分答案分！

已知全級的平均分 $\bar{x} = 50$ 分，標準差 $\sigma = 4$。

代入小英的分數 $x = 50$：

小英的標準分 $z = \frac{50 - 50}{4}$ 1M

$z = \frac{0}{4}$

小英的標準分 $\mathbf{= 0}$ 1A

⚠️ 這是很多同學失分的關鍵題型！必須逐步推論。

首先評估「剔除小英分數」對平均分的影響：

  • 小英的分數是 $50$ 分，而全級的平均分剛好也是 $50$ 分。
  • 結論： 剔除一個「等於平均分」的數據，並不會改變剩餘數據的平均分。
剔除小英的分數後，
全級測驗的平均分不變 (維持 $50$ 分)。 1M

接著評估對標準差的影響：

因為少了一個剛好等於平均分的數據（即離差為 $0$、最不分散的數據），剩下來的數據整體而言顯得相對更分散了。
因此，新的標準差 ($\sigma_{new}$) 會變大 (大於原本的 $4$)。

最後，重新計算小平的新標準分 ($z_{new}$)：

小平的新標準分 $z_{new} = \frac{58 - \text{新平均分}}{\text{新標準差}}$

/* 代入：新平均分仍是 50 */
$z_{new} = \frac{58 - 50}{\sigma_{new}} = \frac{8}{\sigma_{new}}$

/* 因為分子 $8$ 為正數，而分母 $\sigma_{new}$ 變大 */
正數除以一個更大的正數，其值必定變小。
所以， $z_{new} < 2$ (舊標準分)
💡 進階邏輯陷阱： 標準分減少的前提是「該標準分為正數」。
標準差變大，代表所有數據的標準分絕對值都會變小（更靠近 $0$）。因為小平原本的標準分是 $+2$ (正數)，靠近 $0$ 代表數值減少。
（反之，若某同學原標準分為 $-2$，當絕對值變小變成 $-1.5$ 時，數值反而是增加的！）
$\because$ 標準分確實減少了
$\therefore$ 同意小平的宣稱。 1A f.t. (基於合理推論)
💡 數學短證：為何剔除平均值，標準差必定變大？

設原有一組數據共 $n$ 個，平均值為 $\mu$。方差公式為：

$\sigma_{old}^2 = \frac{\sum (x_i - \mu)^2}{n}$

剔除的數據 $x_k = \mu$，它對分子離差平方和的貢獻為 $(x_k - \mu)^2 = 0$。

剔除後，數據個數變為 $n-1$。新的分子因為減去的是 $0$，所以保持不變。新方差為：

$\sigma_{new}^2 = \frac{\sum_{i \neq k} (x_i - \mu)^2}{n - 1} = \frac{\text{原來的分子}}{n - 1}$

分子不變，分母變小 $\Rightarrow$ 分數值變大。
$\therefore \sigma_{new}^2 > \sigma_{old}^2$ (方差變大，標準差也變大)
