題目

圓 $C$ 的方程為 $x^2 + y^2 - 24x - 32y - 276 = 0$。設 $G$ 為 $C$ 的圓心，原點記為 $O$。

  1. 求 $G$ 的坐標。
  2. 求 $OG$ 的長度。
  3. 設 $P$ 為動點使得 $OP = GP$，軌跡為 $\Gamma$。
    $\Gamma$ 與 $C$ 交於 $M$ 及 $N$。求四邊形 $OMGN$ 的面積。
  4. 若過 $A(0, -10)$ 的直線 $L$ 是圓 $C$ 的切線，求 $L$ 的所有可能之方程。

(a) 尋找圓心 $G$

$$ G = \left( -\frac{-24}{2}, -\frac{-32}{2} \right) = (12, 16) $$

(b) 求 $OG$ 的長度

$$ OG = \sqrt{12^2 + 16^2} = 20 \text{ 單位} $$

(c) 軌跡與四邊形面積

半徑 $r = \sqrt{12^2 + 16^2 - (-276)} = 26$。
$\Gamma$ 是 $OG$ 的垂直平分線。面積為 $480 \text{ 平方單位}$。

(d) 求切線 $L$ 的方程

4 分

方法一：傳統代入法 ($\Delta = 0$)

設切線 $L$ 的斜率為 $m$，過 $A(0, -10)$ 的直線方程為：
$y - (-10) = m(x - 0) \implies y = mx - 10$

將 $y = mx - 10$ 代入圓 $C$ 的方程，整理成一元二次方程：

$$ \begin{aligned} x^2 + (mx - 10)^2 - 24x - 32(mx - 10) - 276 &= 0 \\ % 中間整理省略以突顯核心步驟 (1 + m^2)x^2 - (52m + 24)x + 144 &= 0 \end{aligned} $$

► 判別式展開詳細步驟：

因 $L$ 是切線，判別式 $\Delta = 0$：

$$ \begin{aligned} \Delta &= b^2 - 4ac = 0 \\[8pt] \big( -(52m + 24) \big)^2 - 4(1 + m^2)(144) &= 0 \\[8pt] % 負號平方後變正，並將 4*144=576 乘開 (52m + 24)^2 - 576(1 + m^2) &= 0 \\[8pt] % 展開完全平方 (a+b)^2 = a^2 + 2ab + b^2 (52m)^2 + 2(52m)(24) + (24)^2 - 576 - 576m^2 &= 0 \\[8pt] 2704m^2 + 2496m + 576 - 576 - 576m^2 &= 0 \\[8pt] 2128m^2 + 2496m &= 0 \\[8pt] 16m(133m + 156) &= 0 \end{aligned} $$

解得 $m = 0$ 或 $m = -\frac{156}{133}$。

方法二：圓心距離公式 ($d = r$) 捷徑

💡 原理短證：點到直線距離 \( d = \frac{|Ax_1 + By_1 + C|}{\sqrt{A^2 + B^2}} \) (點擊展開詳細證明)

設點 $P(x_1, y_1)$ 及直線 $L: Ax + By + C = 0$。斜率 $m_L = -\frac{A}{B}$。

從 $P$ 點作一直線 $L'$ 垂直於 $L$ 交於點 $Q(x_2, y_2)$，該距離 $d = PQ$。

因 $L' \perp L$，所以 $L'$ 的斜率為 $\frac{B}{A}$。由此可得：

$$ \frac{y_2 - y_1}{x_2 - x_1} = \frac{B}{A} \implies \frac{x_2 - x_1}{A} = \frac{y_2 - y_1}{B} = k $$

即 $x_2 = x_1 + Ak$ 及 $y_2 = y_1 + Bk$。將 $Q(x_2, y_2)$ 代入 $L$ 的方程：

$$ \begin{aligned} A(x_1 + Ak) + B(y_1 + Bk) + C &= 0 \\ Ax_1 + A^2k + By_1 + B^2k + C &= 0 \\ k(A^2 + B^2) &= -(Ax_1 + By_1 + C) \\ k &= \frac{-(Ax_1 + By_1 + C)}{A^2 + B^2} \end{aligned} $$

最後利用兩點距離公式計算 $d = PQ$：

$$ \begin{aligned} d &= \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} \\ &= \sqrt{(Ak)^2 + (Bk)^2} \\ &= \sqrt{k^2(A^2 + B^2)} = |k|\sqrt{A^2 + B^2} \\ &= \left| \frac{-(Ax_1 + By_1 + C)}{A^2 + B^2} \right| \times \sqrt{A^2 + B^2} \\ d &= \frac{|Ax_1 + By_1 + C|}{\sqrt{A^2 + B^2}} \quad \text{(證畢)} \end{aligned} $$

將切線 $L: y = mx - 10$ 寫成一般式：$mx - y - 10 = 0$。
圓心 $G(12, 16)$ 至切線的垂直距離 $d$ 等於半徑 $r = 26$：

$$ \begin{aligned} \frac{|Ax_1 + By_1 + C|}{\sqrt{A^2 + B^2}} &= 26 \\ \frac{|m(12) - (1)(16) - 10|}{\sqrt{m^2 + (-1)^2}} &= 26 \\ \frac{|12m - 26|}{\sqrt{m^2 + 1}} &= 26 \\ |12m - 26| &= 26\sqrt{m^2 + 1} \end{aligned} $$

左右兩邊同時取平方 (Squaring both sides)：

$$ \begin{aligned} (12m - 26)^2 &= 26^2(m^2 + 1) \\ 144m^2 - 624m + 676 &= 676m^2 + 676 \\ -532m^2 - 624m &= 0 \\ -4m(133m + 156) &= 0 \end{aligned} $$

同樣解得 $m = 0$ 或 $m = -\frac{156}{133}$。

最終結論：

將 $m$ 代回 $y = mx - 10$，切線的所有可能方程為：
當 $m = 0$ 時：$y = -10$
當 $m = -\frac{156}{133}$ 時：$y = -\frac{156}{133}x - 10 \implies$ $156x + 133y + 1330 = 0$
