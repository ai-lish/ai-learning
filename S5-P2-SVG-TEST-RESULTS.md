## 根本原因

HTML 中有 **15 個 SVG 容器**，但淨係有 **5 種不同既 viewBox**：

| viewBox | 出現次數 | 對應題目 |
|---------|----------|----------|
| 0 0 400 300 | 10次 | Q9,16,18,19,22,26,28 等 |
| -10 -10 440 330 | 1次 | Q17 |
| 0 0 800 800 | 1次 | Q27 |
| 0 0 400 350 | 1次 | Q29 |
| 0 0 800 400 | 1次 | Q30 |

**問題**：所有 viewBox="0 0 400 300" 既 SVG 都顯示相同既 Q9 內容！
