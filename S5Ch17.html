17.1 計數原理

➕ 加法法則

n₁ + n₂ + ...
「或者」OR

✖️ 乘法法則

n₁ × n₂ × ...
「和」AND
判斷題型：

睇關鍵字 → 「或者」→ 加法 → 「和」→ 乘法

加法法則：完成工作有 k 種互斥方法，總數 = n₁ + n₂ + ... + nₖ

乘法法則：工作由 k 個連續步驟組成，總數 = n₁ × n₂ × ... × nₖ

例：會所雜誌

6本體育、4本財經、7本旅遊

(i) 揀1本：6+4+7 = 17 (加法)

(ii) 每類各1本：6×4×7 = 168 (乘法)

17.2 排列 vs 17.3 組合

🔢 排列 P - 要考慮次序

ABC ≠ BAC

$$P^n_r = \frac{n!}{(n-r)!}$$

例：跑步名次、密碼

📦 組合 C - 唔考慮次序

{A,B,C} = {B,A,C}

$$C^n_r = \frac{n!}{r!(n-r)!}$$

例：抽獎、選隊伍

快速判斷：

改變次序會唔會變？ → 會 → P 排列 → 唔會 → C 組合

排列 (Permutation)：選取物件時，次序影響結果。

排列公式： Pⁿᵣ = n! / (n-r)!

組合 (Combination)：選取物件時，次序不影響結果。

組合公式： Cⁿᵣ = n! / [r!(n-r)!] = Pⁿᵣ ÷ r!

例：4人選3人

排列：ABC, ACB, BAC, BCA, CAB, CBA = 6種

組合：{ABC} = 1種

📌 公式速記

階乘

n! = n×(n-1)×...×1

排列

P = 排完仲要計

組合

C = P ÷ r!

由排列推導組合：

排列 Pⁿᵣ = n!/(n-r)!

每個組合有 r! 種排列

所以 Cⁿᵣ = Pⁿᵣ ÷ r! = n! / [r!(n-r)!]

(i) 3+5=8 (ii) 3×5×4=60
