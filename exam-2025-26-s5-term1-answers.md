### Q3. 主項變換 (3分)
**題目：** 令 $k$ 成為公式 $\frac{k}{h} + \frac{1}{3h} = k$ 的主項。

**解答步驟：**
1. $\frac{k}{h} + \frac{1}{3h} = k$
2. 合併左邊：$\frac{3k+1}{3h} = k$
3. 兩邊乘 $3h$：$3k + 1 = 3hk$
4. 移項：$1 = 3hk - 3k$
5. 提取公因數 $3k$：$1 = 3k(h - 1)$
6. **$k = \frac{1}{3(h - 1)}$**

**評分標準：**
- 步驟1-2：1分
- 步驟3：1分
- 步驟4-6：1分

### Q22. 軌跡與圓 (11分)
**題目：** 圓 $C$ 的方程為 $x^2 + y^2 - 24x - 32y - 276 = 0$。設 $G$ 為圓心，$O$ 為原點。

**(a) 求 $G$ 的坐標** (1分)
- 圓心公式：$(-\frac{D}{2}, -\frac{E}{2}) = (-\frac{-24}{2}, -\frac{-32}{2})$
- **$G = (12, 16)$**

**(b) 求 $OG$** (2分)
- $OG = \sqrt{12^2 + 16^2} = \sqrt{144 + 256} = \sqrt{400} = \mathbf{20}$

**(c) 求四邊形 $OMGN$ 的面積** (4分)
- 軌跡 $\Gamma$ 為 $OG$ 的垂直平分線
- $K$ 為 $OG$ 中點：$K = (6, 8)$
- 圓 $C$ 半徑：$r = \sqrt{12^2 + 16^2 - (-276)} = \sqrt{676} = 26$
- $GK = 10$，在直角三角形 $GKN$ 中：
- $KN = \sqrt{26^2 - 10^2} = \sqrt{676 - 100} = \sqrt{576} = 24$
- $MN = 2 \times 24 = 48$
- 面積 $= \frac{1}{2} \times OG \times MN = \frac{1}{2} \times 20 \times 48 = \mathbf{480}$

**(d) 求切線方程** (4分)
- 設切線 $L$：$y + 10 = mx$ 即 $mx - y - 10 = 0$
- 圓心到切線距離 = 半徑：
- $\frac{|12m - 16 - 10|}{\sqrt{m^2 + 1}} = 26$
- $|12m - 26| = 26\sqrt{m^2 + 1}$
- $|6m - 13| = 13\sqrt{m^2 + 1}$
- $(6m - 13)^2 = 169(m^2 + 1)$
- $36m^2 - 156m + 169 = 169m^2 + 169$
- $133m^2 + 156m = 0$
- $m(133m + 156) = 0$
- $m = 0$ 或 $m = -\frac{156}{133}$
- **切線方程：$y + 10 = 0$ 或 $156x + 133y + 1330 = 0$**

**評分標準：**
- (a)：1分
- (b)：2分
- (c)：4分
