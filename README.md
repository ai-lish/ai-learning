### T仔測試技巧 (⚠️ 重要)

**原則**：Button存在 ≠ 功能正常！

**正確流程**：
1. 檢查實際 href value
   ```bash
   grep 'href="index.html"' S1Ch1.html
   ```
2. 真正既 Click 測試
   ```bash
   browser action=open url="..."
   browser action=act kind=click ref=e4
   ```
3. 驗證URL變化
