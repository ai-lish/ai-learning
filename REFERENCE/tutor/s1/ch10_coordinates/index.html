📗 Ch10 坐標簡介

中一數學 | 坐標系統、象限、距離、面積、變換

📖 重點筆記

1. 坐標系統

坐標用來表示平面上一點的位置。

格式：(x, y)
先寫 X（橫座標），再寫 Y（豎座標）
中間用逗號分隔，外加括號
例如：P(3, 4) → x = 3, y = 4
點 (2, 5) 表示：

• 向右走 2 格

• 向上走 5 格

2. 象限

坐標平面被 X 軸和 Y 軸分成四個區域，稱為「象限」。

第一象限
x > 0, y > 0
(+,+)
第二象限
x < 0, y > 0
(-,+)
第三象限
x < 0, y < 0
(-,-)
第四象限
x > 0, y < 0
(+,-)
例：點 (3, -2) 在第四象限

因為 x 為正數，y 為負數

3. 鉛垂距離與水平距離

兩點之間的水平和鉛垂距離：

水平距離 = |x₂ - x₁|
鉛垂距離 = |y₂ - y₁|
例：求 A(2, 3) 到 B(6, 7) 的距離

水平距離 = |6 - 2| = 4

鉛垂距離 = |7 - 3| = 4

4. 求面積

可以用「割補法」求坐標圖形面積

例：求三角形 ABC 的面積
A(1,1), B(5,1), C(3,4)

底 = 4 (由 x=1 到 x=5)

高 = 3 (由 y=1 到 y=4)

面積 = 4 × 3 ÷ 2 = 6 平方單位

5. 變換

📐 平移 (Translation)

將點向指定方向移動

P(x,y) 向右移 a 單位 → P'(x+a, y)

P(x,y) 向上移 b 單位 → P'(x, y+b)

🪞 反射 (Reflection)

關於軸對稱地反射

關於 X 軸：P(x,y) → P'(x, -y)

關於 Y 軸：P(x,y) → P'(-x, y)

🔄 旋轉 (Rotation)

繞原點旋轉

逆時針 90°：P(x,y) → P'(-y, x)

順時針 90°：P(x,y) → P'(y, -x)

📝 測驗題目

Question 1 (MC)

寫出點 ( -4, -3 ) 的位置：

A. x = -4, y = -3    B. x = -3, y = -4    C. x = 4, y = 3    D. x = 3, y = 4

答案：A. x = -4, y = -3
解說：坐標格式 (x, y)，第一個係 X，第二個係 Y

Question 2 (MC)

點 (2, -5) 在第幾象限？

A. 第一象限    B. 第二象限    C. 第三象限    D. 第四象限

答案：D. 第四象限
解說：x > 0, y < 0 屬於第四象限

Question 3 (MC)

將點 (3, 4) 關於 X 軸反射，結果係邊一點？

A. (-3, 4)    B. (3, -4)    C. (-3, -4)    D. (4, 3)

答案：B. (3, -4)
解說：關於 X 軸反射：y 座標變號，x 座標不變

Question 4 (Long Question)

三角形 ABC 的頂點分別為 A(1,1)、B(5,1)、C(5,4)。

(a) 呢個三角形係咩三角形？

(b) 計算面積

答案：
(a)直角三角形（因為有一邊垂直於 X 軸）
(b)底 = 4，高 = 3，面積 = 4 × 3 ÷ 2 = 6 平方單位
